# Sales System Project

Projeto standalone do sistema de vendas.

## Rodar localmente

```bash
yarn install
yarn db:setup
yarn dev
```

Aplicação: `http://localhost:3002`

## Observações

- este diretório foi preparado para virar um repositório próprio
- crie `.env` a partir de `.env.example` com as URLs do Neon/Postgres
- o client Prisma gerado e a pasta `.next` devem ser recriados localmente
