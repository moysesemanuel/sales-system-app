# Sales System Project

Projeto standalone do sistema de vendas.

## Rodar localmente

```bash
yarn db:setup
yarn dev
```

Aplicação: `http://localhost:3002`

## Observações

- este diretório foi preparado para virar um repositório próprio
- o arquivo `.env` não foi copiado, apenas `.env.example`
